# Getting Started

The library creates a simple and intuitive method of routing your application's URLs
based on your file and folder structure.

## Install

<Tabs>
  <TabItem value="npm" label="npm" default>
    <CodeBlock language="bash">
      npm install node-file-router
    </CodeBlock>
  </TabItem>
  <TabItem value="yarn" label="Yarn">
    <CodeBlock language="bash">
      yarn add node-file-router
    </CodeBlock>
  </TabItem>
  <TabItem value="pnpm" label="pnpm">
    <CodeBlock language="bash">
      pnpm add node-file-router
    </CodeBlock>
  </TabItem>
</Tabs>

### What you'll need

- [Node.js](https://nodejs.org/en/download/) version 16 or above
